**Stop reading 10-page assignment docs. Just chat with them.**

Classroom Bot is a Chrome Extension that integrates directly into Google Classroom. It reads the active assignment document and allows you to ask questions about it instantly. It acts as a personal teaching assistant, clarifying requirements, due dates, and criteria without you having to scroll endlessly.
## üöÄ Features

* **Context-Aware Q&A:** Answers questions based *strictly* on the document currently open in your tab.
* **Seamless Integration:** Embeds a chat interface directly into the Google Classroom page.
* **Instant Summaries:** Get quick overviews of long PDFs or Google Docs.
* **Hallucination-Free:** Designed to prioritize facts found within the text, reducing AI errors.

## üì¶ Installation

Since this extension is in development/beta, you will need to load it manually into Chrome ("Developer Mode").

1.  **Clone this repository:**
    ```bash
    git clone [https://github.com/dharmik2212/Google-classroom--Extenstion](https://github.com/dharmik2212/Google-classroom--Extenstion)
    ```
2.  Open Google Chrome and navigate to `chrome://extensions/`.
3.  Toggle **Developer mode** in the top right corner.
4.  Click **Load unpacked** in the top left.
5.  Select the folder where you cloned this repository.
6.  The extension is now installed! Pin it to your toolbar for easy access.
